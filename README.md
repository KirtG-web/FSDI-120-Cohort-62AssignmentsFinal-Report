Hungarian Heritage Website

This is my final project for the Full Stack Development Immersive program.

ABOUT THE PROJECT
This website was created to showcase Hungarian culture, history, and traditions.
It was also built as a personal tribute to my wife, Erzsébet Bérczi German, who inspired my connection to Hungary.

TECH STACK

* Python / Django
* HTML / CSS / Bootstrap
* SQLite
* GitHub
* Render (deployment)

FEATURES

* Multi-page responsive design
* Cultural content (food, history, language, etc.)
* Translation support (English / Hungarian)
* Video and image integration
* Contact page

HOW TO RUN LOCALLY

1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Run server

Commands:

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

Then go to: http://127.0.0.1:8000/

DEPLOYMENT
This project is deployed using Render.
